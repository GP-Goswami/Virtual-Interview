## Overview
Virtual Interview is an AI-based interview preparation tool designed to help students and job seekers practice real interview scenarios.  
The application aims to remove hesitation, improve communication confidence, and provide a realistic interview experience through an AI virtual assistant.

###  How It Works
1. Upload your resume  
2. Click the **Start Interview** button  
3. A confirmation message appears  
4. After clicking **OK**, the interview begins  
5. The AI assistant starts asking questions  
6. User and AI communicate in real-time during the interview  

This gives users an experience similar to facing a real interviewer.

## Installation

### 1. Create a Virtual Environment
`python -m venv venv`

### 2. Activate the Environment
#### Windows:
      `venv\Scripts\activate`

#### Linux / macOS:

      `source venv/bin/activate`

### 3. Install Dependencies
      `pip install -r requirements.txt`

---


## Usage
### 1. Start the Streamlit Frontend
`streamlit run app.py`

### 2. Start the FastAPI Backend
`uvicorn main:app --reload`

### 3. Open the Application

Streamlit will automatically open in the browser.
If not, go to:

`http://localhost:8501`
